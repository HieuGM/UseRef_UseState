So sánh useRef và useState trong React

Tiêu chí useRef useState
Giống nhau
Sử dụng trong function component Cả "useRef" và "useState" đều được sử dụng trong function component của React.
Khác nhau
Cập nhật giá trị Cập nhật giá trị không làm component render lại. Cập nhật giá trị làm component render lại.
Lưu trữ giá trị Lưu trữ giá trị tham chiếu (có thể thay đổi), có thể giữ qua các lần render mà không kích hoạt re-render. Lưu trữ giá trị (không thể thay đổi), giá trị sẽ được lưu giữ và thay đổi qua các lần render.
Truy cập DOM Thường dùng để tham chiếu trực tiếp đến phần tử DOM. Không được dùng để tham chiếu đến DOM, chỉ dùng để lưu trạng thái của component.
Sử dụng Khi cần lưu trữ giá trị qua các lần render mà không cần re-render (ví dụ: đếm số lần render, lưu trữ giá trị input trước đó). Khi cần giá trị làm thay đổi UI và cần re-render khi giá trị thay đổi (ví dụ: quản lý trạng thái như toggle, form input).
